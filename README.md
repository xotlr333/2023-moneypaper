## 💌 서비스 소개

`머니페이퍼`는 우리FISA 개인 방명록 커뮤니티입니다.

우리FISA 교육생분들의 원활한 네트워킹을 위해 익명으로 다른 사람의 방명록에 글을 남기는 서비스입니다.
